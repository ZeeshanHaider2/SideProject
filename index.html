About

A shopping cart made on vanillaJS(ES6) by using browser as internel storage. The user can add items into the cart, edit the quantity of items in the cart and can also delete the whole cart. There is also a payment option and the user can choose to pay for the items by using paypal. Also, the user interface is fully responsive i.e.can be seen on screens of different sizes, for example: Laptops,tablets and mobiles.
